Cybersecurity Advent - Day 20

2017-12-20 By Teejay

Today's Cybersecurity tip is: use HTTPS everywhere

 I've already touched on encryption in this series, so you're aware of the importance of scrambling your data. I've also briefly touched on encryption of your data in transit using HTTPS.

So today's tip is a quick and easy one in which you install the HTTPS Everywhere browser plugin provided by the Electronic Frontier Foundation. 

This plugin will automatically redirect your browser traffic to secure, HTTPS-enabled servers if available or if their ruleset is in place. While not perfect, and doesn't cover every site on the internet, it still adds another level of security to your day-to-day browsing. 

That's it ... that's all I've got today. Even though it's short, it's still important; so go install that plugin now.

See you tomorrow!
